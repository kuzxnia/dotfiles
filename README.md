# dotfiles

To setup full configurations like ZSH, git, neovim and other stuff run:
```bash
python3 -c "$(wget https://raw.githubusercontent.com/kuzxnia/dotfiles/master/manage.py -O -)" full_setup
```

Other options:
* full_setup
* IDE

* vim
* neovim
* tmux
* kitty
* rust
* zsh
* libs
* external_libs
