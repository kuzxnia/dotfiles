# git basics

```bash

git diff # modification in working copy in console
git difftool # in external editor

git tag # list of tags
git tag [name] [optional commit hash] # sign commit with tag (lightweight - tag name)
git tag [name] [optional commit hash] -a -m 'msg' # annotated tag (stores also author, created date)
git tag -d [name]

git show [tag_name\commit hash] # show commit information
```
